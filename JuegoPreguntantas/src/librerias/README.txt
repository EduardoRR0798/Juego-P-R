	README para la instalación de librerías
	==============================================

Este demo ilustra como instalar librerías en el proyecto a traves de NetBeans IDE 8.2.

Contenido
------------

-activation.jar (JavaBeans (tm) Activation Framework 1.1.1)
-mail.jar (JamaMail (TM) API 1.4.7)

Instalación
------------

1. Click derecho en Libraries

2. Click en Add JAR/Folder...

3. Seleccionar activation.jar y mail.jar

4. Click en Abrir
