	README para la instalación de librerías
	==============================================

Este demo ilustra como instalar librerías de Oracle en el proyecto a traves de NetBeans IDE 8.2.

Contenido
------------

-activation.jar (JavaBeans (tm) Activation Framework 1.1.1)
-mail.jar (JamaMail (TM) API 1.4.7)
-dsn.jar
-gimap.jar
-imap.jar
-mailapi.jar
-pop3.jar
-smtp.jar

Instalación
------------

1. Click derecho en Libraries

2. Click en Add JAR/Folder...

3. Seleccionar todos los jar de la carpeta

4. Click en Abrir
